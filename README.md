# node-simplex-noise

> A modern, node-based implementation of Simplex Noise

Based on [Stefan Gustavson's implementation](http://staffwww.itn.liu.se/~stegu/simplexnoise/simplexnoise.pdf) of Ken Perlin's Simplex Noise algorithm.
